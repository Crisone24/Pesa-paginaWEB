# Pesa-paginaWEB
Diseño de bascula con pagina web y esp32 con pantalla tactil que muestran el peso de los objetos, la idea principal no es el peso sino la deformación que se genera en la galga.


Librerias a usar:

--> ArduinoJson

--> HX711_Arduino_Library

--> LiquidCrystal

-->LovyanGFX

-->lvgl
